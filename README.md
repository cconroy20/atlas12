# atlas12
Bob Kurucz's ATLAS12 program for stellar atmospheres, translated into modern F90

# input data
All necessary input data should reside in the data directory.  There are a seven large binary files that must be downloaded separately.  These are: 
gfpred29dec2014.bin
schwenke.bin
lowobsat12.bin
gfall29dec2014.bin
diatomicspacksrt.bin
hilines.bin
h2ofastfix.bin
